What?

Billtab is a tool to collaboratively track and analyse expenses and earnings. The collaborative features are in development but the current version works well for personal use and offline.

Get Started

Entries are currently locally stored on your system so that they can be available offline. No data is currently shared with the Billtab server.

Made by Opeyemi Obembe. Source available at github.com/kehers/billtab.
